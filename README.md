## Contents

### Notebook 01: First Look and Discovery
- Ran a complete machine learning pipeline using anonymized search data.
- Compared a hand-written rule with a machine learning model.

### Notebook 02: Your First Readable Model

- Trained a decision tree model and interpreted its decisions.
- Compared model performance with a manual approach.
