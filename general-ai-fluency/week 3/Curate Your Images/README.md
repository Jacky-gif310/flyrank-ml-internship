# Curate Your Images — Jackline Mutheu

## Final Image Set

**AI-generated (decorative only)**

- Hero background: Minimal trend-line graphic using my Identity Kit colors (#2B4C5C, #14161A, #F7F8FA, #D8A24A).
- Icons (all created in one consistent minimalist line style):
  - Notebook/document — Projects section
  - Trend/chart — Machine Learning and Analytics sections
  - Review/checkmark — Completed projects, especially Refresh / Content Opportunity Scoring
  - Data node (used sparingly) — Dataset and research sections only

**Real images (proof of work)**

- GitHub repository screenshots
- Jupyter Notebook screenshots
- Charts generated from my notebook analyses
- Research paper and case study screenshots

## Where I Chose Real Images Instead of AI

I used real screenshots for my GitHub repositories, Jupyter notebooks, charts, and research documents because they provide verifiable evidence of my machine learning work — a generated stand-in would undercut the "proof" the portfolio is meant to demonstrate. AI-generated images are used only for decorative elements such as the hero background and icons, never for anything that claims to show actual work or results. This portfolio doesn't include a personal bio photo section, so no photo was needed for this deliverable.

## Rejected Images

**Network/node graph**
Although visually interesting, this is a common AI design cliché that suggests "artificial intelligence" in a generic way rather than communicating my actual work. The trend-line graphic better reflects search analytics, machine learning, and research while keeping the focus on real project evidence rather than decoration.

**Scatter field**
The scatter field looked decorative but had no clear purpose without labels or context, and it risked confusing visitors about what it represented. The trend-line illustration communicates data analysis and progress immediately, making it a stronger visual choice.

## Style Consistency

All AI-generated graphics follow the same minimalist editorial style and use my Identity Kit palette (#2B4C5C, #14161A, #F7F8FA, and #D8A24A). The visual design remains calm and understated so that the portfolio emphasizes the machine learning projects and research rather than decorative artwork.
